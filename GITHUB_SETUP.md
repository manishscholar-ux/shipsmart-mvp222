# ShipSmart MVP - GitHub Setup Instructions

## Step 1: Create GitHub Account
Go to https://github.com/signup and create your free account.

## Step 2: Create New Repository
1. Go to https://github.com/new
2. Repository name: `shipsmart-mvp`
3. Description: "Shipping Label Management Platform"
4. Choose "Public"
5. Click "Create repository"

## Step 3: Upload Your Files
In your new GitHub repository:
1. Click "Add file" → "Upload files"
2. Drag and drop ALL files from: `c:\Users\YadavManish\OneDrive - Purolator Inc\Desktop\test codes`
3. Add commit message: "Initial commit - ShipSmart MVP"
4. Click "Commit changes"

## Files to Upload (All files in your project folder):
- index.html ✅
- package.json
- tsconfig.json
- next.config.mjs
- .gitignore (create if needed)
- app/ (entire folder)
- components/ (entire folder)
- lib/ (entire folder)
- public/ (entire folder)
- prisma/ (entire folder)
- scripts/ (entire folder)

## Step 4: Deploy to Vercel
1. Go to https://vercel.com/sign-up
2. Click "Continue with GitHub"
3. Click "Import Project"
4. Select your `shipsmart-mvp` repository
5. Vercel auto-detects Next.js
6. Click "Deploy"

## Step 5: Wait & Share
- Vercel deploys automatically (2-3 minutes)
- Your app goes live at: `https://shipsmart-mvp.vercel.app`
- Share this URL with anyone!
