## **TROUBLESHOOTING**

### "Upload is taking too long"
- Try uploading fewer files at once
- Or use Git command line instead (see bottom)

### "Vercel deployment failed"
- Check Vercel "Deployments" tab for error
- Most common: wait a bit, try again
- Click "Redeploy" button

### "App shows blank or errors"
- Wait 1-2 more minutes (still building)
- Refresh browser (Ctrl+Shift+R hard refresh)
- Check Vercel logs for clues

### "Can't access GitHub"
- Try incognito/private browser window
- Clear cookies
- Try different browser
