## **Support During Deployment**

### If something goes wrong:

1. **GitHub upload failing?**
   - Try uploading fewer files at once
   - Or use: git command line (see below)

2. **Vercel deployment failing?**
   - Check Vercel "Deployments" tab for error logs
   - Most common: missing files, syntax error
   - Click "Redeploy" to try again

3. **App not loading after deploy?**
   - Wait 1-2 more minutes
   - Refresh browser (Ctrl+Shift+R)
   - Check Vercel logs
